## 🚀 Installation & Development

To clone, set up, and launch this portfolio website locally, follow the steps below:

### 📦 Setup Environment
Ensure [Node.js](https://nodejs.org/) (version 18+) is installed on your workstation.

```bash
# 1. Clone the repository
git clone https://github.com/lokanathmeher19/lokanathmeher19.github.io.git
cd lokanathmeher19.github.io

# 2. Install modern dependencies
npm install
```

### ⚙️ Add Local Credentials
Create a `.env` file inside the root directory to support the contact email submission features:
```ini
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 💻 Run Development Server
```bash
npm run dev
```
The console will display the local address. Typically: `http://localhost:8000/`.
