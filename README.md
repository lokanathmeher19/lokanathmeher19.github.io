## 🛠️ Built With

This portfolio is built using a modern frontend stack to ensure high performance, smooth animations, and clean design.

- **Frontend:** React.js, Vite, Tailwind CSS, Framer Motion
- **Interactions:** EmailJS (for contact form integration)
- **Deployment:** GitHub Pages

## ⚙️ Local Development & Setup

To clone, configure, and run this portfolio website locally:

### 1. Clone & Install
```bash
# Clone the repository
git clone https://github.com/lokanathmeher19/lokanathmeher19.github.io.git

# Navigate to the project directory
cd lokanathmeher19.github.io

# Install dependencies
npm install
```

### 2. Environment Configuration
Create a `.env` file in the root directory to enable contact form functionality:
```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

### 3. Run Development Server
```bash
npm run dev
```
The application will run locally at `http://localhost:8000/` (or check console output for the exact port).
